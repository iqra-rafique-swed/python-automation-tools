# Python Automation Tools

This repository contains simple Python automation scripts created for learning and practice.

## Scripts

### 1. File Renamer
A script that automatically renames all files in a folder.

**How to use:**
- Update the folder path inside the script.
- Run the script.
- All files will be renamed in sequence.

### 2. CSV Cleaner
A script that removes empty rows from a CSV file.

**How to use:**
- Update the input and output file names in the script.
- Run the script.
- A cleaned CSV file will be generated.

### 3. Image Converter
A script that converts images from one format to another (e.g., JPG to PNG).

**How to use:**
- Set the input and output folder paths.
- Choose the target format (default: PNG).
- Run the script to convert all images.
